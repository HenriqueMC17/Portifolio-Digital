# Portfólio Digital — Henrique Monteiro Cardoso

Portfólio profissional desenvolvido em **Next.js + React + TypeScript**, com foco em apresentação técnica, performance, responsividade e experiência de navegação consistente.

## Descrição

Este projeto apresenta:

- Posicionamento profissional (Hero e proposta de valor)
- Seções de experiência, projetos, habilidades e certificações
- Assistente virtual (HenriqueBot)
- Formulário de contato com validação em tempo real
- Estrutura visual dark-first e componentes reutilizáveis

## Tecnologias utilizadas

- Next.js 15
- React 19
- TypeScript 5
- Tailwind CSS
- Framer Motion
- Radix UI (primitivos de interface)

## Execução local

### Pré-requisitos

- Node.js 18+
- pnpm

### Instalação

```bash
pnpm install
```

### Ambiente de desenvolvimento

```bash
pnpm dev
```

### Validações técnicas

```bash
pnpm lint
pnpm audit:portfolio
pnpm -s tsc --noEmit
```
