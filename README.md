## 🌟 Visão Geral

Este portfólio foi desenvolvido para apresentar minha jornada profissional, habilidades técnicas e projetos de forma interativa e visualmente atraente. O site é totalmente responsivo, oferece suporte a múltiplos idiomas (Português e Inglês) e inclui animações avançadas para melhorar a experiência do usuário.

### 🎯 Objetivos do Projeto

- **Showcase Profissional**: Apresentar habilidades e experiências de forma impactante
- **Experiência Interativa**: Navegação envolvente e moderna
- **Performance Otimizada**: Carregamento rápido e experiência fluida
- **Acessibilidade**: Melhores práticas de acessibilidade web
- **SEO Otimizado**: Estrutura otimizada para mecanismos de busca

## 📁 Estrutura do Projeto

```text
portfolio-website/
├── app/                # App Router do Next.js
│   ├── page.tsx        # Página principal
│   ├── layout.tsx      # Layout raiz
│   ├── globals.css     # Estilos globais
│   └── api/            # API Routes
├── components/         # Componentes React
│   ├── sections/       # Seções principais
│   └── ui/             # Componentes de UI
├── contexts/           # Contextos React
├── hooks/              # Hooks personalizados
├── lib/                # Utilitários
├── locales/            # Traduções
├── public/             # Arquivos estáticos
└── config files        # Configuração (next, tailwind, tsconfig, etc)
```

## 🚀 Como Executar

### Pré-requisitos

- Node.js 18.18.0+
- npm, yarn, pnpm ou bun
- Git

### Instalação

```bash
git clone https://github.com/HenriqueMC17/portfolio-website.git
cd portfolio-website
npm install
npm run dev
```

Acesse: [http://localhost:3000](http://localhost:3000)

## 🔧 Scripts Disponíveis

```bash
npm run dev          # Desenvolvimento
npm run build        # Build de produção
npm run start        # Produção
npm run lint         # Lint
npm run format       # Prettier
npm run type-check   # TypeScript
npm run analyze      # Bundle size
npm run lighthouse   # Lighthouse
npm run pwa:generate # Gera ícones PWA
```
