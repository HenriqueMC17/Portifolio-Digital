## 🌟 Visão Geral

Este portfólio foi desenvolvido para apresentar minha jornada profissional, habilidades técnicas e projetos de forma interativa e visualmente atraente. O site é totalmente responsivo, oferece suporte a múltiplos idiomas (Português e Inglês) e inclui animações avançadas para melhorar a experiência do usuário.

### 🎯 Objetivos do Projeto

- **Showcase Profissional**: Apresentar habilidades e experiências de forma impactante
- **Experiência Interativa**: Criar uma navegação envolvente e moderna
- **Performance Otimizada**: Garantir carregamento rápido e experiência fluida
- **Acessibilidade**: Seguir as melhores práticas de acessibilidade web
- **SEO Otimizado**: Estrutura otimizada para mecanismos de busca

## 🎮 Easter Eggs

O portfólio inclui **8 Easter Eggs interativos** que podem ser descobertos através de sequências secretas. Cada Easter Egg oferece uma experiência visual única e divertida!

### 🔓 Como Descobrir

Digite as sequências abaixo em qualquer lugar do site (não funciona em campos de input):

| Easter Egg | Sequência | Efeito |
|------------|-----------|---------|
| 🎊 **Konami Code** | `↑↑↓↓←→←→BA` | Chuva de confetti colorido |
| 🔧 **Modo Desenvolvedor** | `dev` | Painel com informações técnicas do site |
| 🟢 **Modo Matrix** | `matrix` | Transforma o site em estilo Matrix |
| 🕺 **Modo Disco** | `disco` | Cores piscantes estilo discoteca |
| 🌌 **Gravidade Zero** | `grav` | Remove a gravidade dos elementos |
| 💥 **Efeito Thanos** | `thanos` | Desintegra metade dos elementos |
| 🌈 **Modo Arco-íris** | `rainbow` | Cores do arco-íris em todo o site |
| 🎂 **Modo Aniversário** | `bday` | Celebração com balões e confetti |

### 🎛️ Painel de Controle

- **Botão Flutuante** - Ícone de ovo no canto inferior esquerdo
- **Progresso de Desbloqueio** - Acompanhe quantos Easter Eggs você descobriu
- **Controle Individual** - Ative/desative cada Easter Egg
- **Persistência** - Seus descobrimentos são salvos localmente

### 🏆 Sistema de Conquistas

- **Descoberta** - Confetti especial ao desbloquear pela primeira vez
- **Notificações** - Toast com nome e descrição do Easter Egg
- **Contador** - Progresso visual (X de 8 descobertos)
- **Reset** - Opção para resetar todos os efeitos ativos

## 🚀 Como Executar

### 📋 Pré-requisitos

- **Node.js** 18.18.0 ou superior
- **npm**, **yarn**, **pnpm** ou **bun**
- **Git** para clonagem do repositório

### 🔧 Instalação

1. **Clone o repositório:**
\`\`\`bash
git clone https://github.com/HenriqueMC17/portfolio-website.git
cd portfolio-website
\`\`\`

2. **Instale as dependências:**
\`\`\`bash
# npm
npm install

# yarn
yarn install

# pnpm
pnpm install

# bun
bun install
\`\`\`

3. **Configure as variáveis de ambiente (opcional):**
\`\`\`bash
cp .env.example .env.local
\`\`\`

4. **Execute o projeto:**
\`\`\`bash
# npm
npm run dev

# yarn
yarn dev

# pnpm
pnpm dev

# bun
bun dev
\`\`\`

5. **Abra no navegador:**
\`\`\`
http://localhost:3000
\`\`\`

## 📱 PWA Features

### 🔧 Funcionalidades

- **📱 Instalável** - Pode ser instalado como app nativo
- **🔄 Offline First** - Funciona sem conexão com internet
- **🔄 Background Sync** - Sincroniza dados quando online
- **📊 Analytics Offline** - Coleta dados mesmo offline
- **🎨 Splash Screen** - Tela de carregamento personalizada
- **📱 App-like Experience** - Comportamento similar a app nativo

### ⚙️ Configuração

\`\`\`json
{
  "name": "Henrique Monteiro Cardoso - Portfólio",
  "short_name": "HMC Portfólio",
  "display": "standalone",
  "orientation": "portrait",
  "theme_color": "#10b981",
  "background_color": "#121212"
}
\`\`\`

### 📊 Cache Strategy

- **Network First** - Para dados dinâmicos (GitHub API)
- **Cache First** - Para assets estáticos
- **Stale While Revalidate** - Para conteúdo que pode ser atualizado

## 🎨 Design System

### 🎨 Cores

\`\`\`css
/* Tema Claro */
--primary: 142 76% 36%        /* Verde principal */
--secondary: 0 0% 96.1%       /* Cinza claro */
--accent: 0 0% 96.1%          /* Accent neutro */
--background: 0 0% 100%       /* Branco */
--foreground: 0 0% 3.9%       /* Preto suave */

/* Tema Escuro */
--primary: 142 70% 45%        /* Verde mais claro */
--background: 0 0% 3.9%       /* Preto suave */
--foreground: 0 0% 98%        /* Branco */
\`\`\`

### 📝 Tipografia

- **Font Principal**: Inter (Google Fonts)
- **Font Mono**: JetBrains Mono (Google Fonts)
- **Escala**: Fluid typography com clamp()

### 🎭 Animações

- **Duração Base**: 0.3s para micro-interações
- **Easing**: cubic-bezier(0.4, 0, 0.2, 1)
- **Reduced Motion**: Respeita preferências de acessibilidade
